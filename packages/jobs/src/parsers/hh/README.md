# HH.ru Parser - Человекоподобное поведение

## Реализованные меры защиты от детекта

### 1. Случайные задержки

- **Между резюме**: 3-8 секунд
- **Между вакансиями**: 2-5 секунд
- **При чтении страницы**: 2-5 секунд
- **При скролле**: 0.5-1.5 секунды
- **Перед запросом контактов**: 1-2.5 секунды

### 2. Имитация движений мыши

- Плавное перемещение курсора с случайным количеством шагов (5-15)
- Случайные координаты в пределах видимой области

### 3. Имитация скролла

- Плавный скролл с `behavior: 'smooth'`
- Случайное расстояние скролла (300-600px)
- Паузы между скроллами

### 4. Имитация чтения

- Движение мыши как будто читаем текст
- Паузы на "чтение" контента (2-5 секунд)

### 5. Скрытие признаков автоматизации

- Переопределение `navigator.webdriver` → `false`
- Добавление реалистичных плагинов
- Установка языков браузера
- Добавление объекта `chrome.runtime`
- Реалистичный User-Agent
- Viewport как у обычного пользователя (1920x1080)

### 6. Puppeteer Stealth Plugin

- Автоматическое скрытие признаков Puppeteer
- Обход базовых методов детекта автоматизации

### 7. Настройки браузера

- `slowMo: 100` - задержка между действиями
- Отключение флагов автоматизации
- Отключение `AutomationControlled`

## Использование

```typescript
import {
  humanDelay,
  humanMouseMove,
  humanScroll,
  humanRead,
  humanBrowse,
} from "./human-behavior";

// Случайная задержка
await humanDelay(1000, 3000);

// Движение мыши
await humanMouseMove(page);

// Скролл страницы
await humanScroll(page);

// Имитация чтения
await humanRead(page);

// Комплексное поведение (движение + скролл)
await humanBrowse(page);
```

## Рекомендации

1. **Не запускать парсер слишком часто** - делайте перерывы между запусками
2. **Использовать прокси** - для дополнительной анонимности
3. **Варьировать время запуска** - не запускать в одно и то же время
4. **Следить за логами** - проверять, нет ли капчи или блокировок
5. **Использовать реальные cookies** - авторизация через сохраненные куки выглядит естественнее
